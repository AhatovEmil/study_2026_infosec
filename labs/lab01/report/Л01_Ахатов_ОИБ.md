# ответы на контрольные вопросы

## 1. Информация учётной записи пользователя
В `/etc/passwd`: имя, пароль (x), UID, GID, GECOS, домашний каталог, оболочка.
Просмотр: `cat /etc/passwd`, `id имя`, `whoami`.

## 2. Команды терминала (примеры)

| Действие | Команды |
|----------|---------|
| **Справка** | `man ls`, `ls --help`, `info ls`, `whatis ls` |
| **Перемещение** | `cd /путь`, `cd ..`, `cd ~`, `cd -`, `pwd` |
| **Просмотр содержимого** | `ls`, `ls -l`, `ls -a`, `ls -la`, `ls -lh` |
| **Объём каталога** | `du -sh каталог`, `du -h --max-depth=1` |
| **Создание/удаление** | `mkdir каталог`, `touch файл`, `rm файл`, `rm -r каталог` |
| **Права доступа** | `chmod 755 файл`, `chmod u+x файл`, `chown пользователь:группа файл` |
| **История команд** | `history`, `!!`, `!N`, `Ctrl+R` |

## 3. Файловая система (ФС)
**Определение:** способ организации данных на носителе.

**Примеры:**
- **ext4** – стандартная журналируемая ФС Linux.
- **XFS** – высокопроизводительная, для больших файлов.
- **Btrfs** – с поддержкой снапшотов и сжатия.
- **FAT32** – совместимая, макс. файл 4 ГБ.
- **NTFS** – журналируемая, права доступа (Windows).
- **tmpfs** – в оперативной памяти (временные файлы).

## 4. Просмотр смонтированных ФС
`mount`, `findmnt`, `df -hT`, `lsblk -f`.

## 5. Удаление зависшего процесса
1. Найти PID: `ps aux | grep имя` или `top`.
2. Завершить: `kill -TERM <PID>`.
3. Принудительно: `kill -KILL <PID>` (или `-9`).
Альтернативы: `pkill имя`, `killall имя`, `htop`.
